Key components

  • tags(标签) (and their attibutes(属性))
    • most commonly in pairs like <h1> and </h1>
  • character-based data types(基于字符的数据类型)
  • character references(字符参数)
  • entity references(实体参数)

document type declaration(文档类型声明): which triggers standards mode rendering.

  • <!DOCTYPE html> is for HTML5
  • If a declaration is not included, various browsers will revert to “quirks mode” for rendering

Elements

  • the extent of an element is indicated by a pair of tags:
  • a “start tag” <p> and “end tag” </p>. The text content of the element, if any, is placed between these tags.
  • Some elements, such as the line break(换行符) <br>, do not permit any embedded content, either text or further tags(不允许任何嵌入的内容包括文本或其他标签).

The general form of an HTML element is therefore:

  • <tag attribute1="value1" attribute2="value2">''content''</tag>
  • Some HTML elements are defined as empty elements and take the form
    • <tag attribute1="value1" attribute2="value2">
    • in empty elements the end tag is neither required nor allowed
