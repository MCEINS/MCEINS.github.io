部署到GitHub

  • create a new repository
  • 链接GitHub和git – ssh
  • 安装Git部署插件
  • 推送

具体步骤

第一步

  • 进入GitHub官网(没有账号去注册一下)
  • 点击账户头像左边+选择New repository 然后到Creat repository页面
  • 仓库名Repository name必须为your_user_name.github.io，我的用户名为MCEINS，所以填写MCEINS.github.io，勾选public公开仓库，并在下面方框打勾，然后点击Creat repository
  • 创建完成后点击settings打开设置
  • 找到GitHub Pages
  • 点击 https://mceins.github.io/ 出现你仓库名字，说明已经生效，可以继续下一步了。

第二步

  • 在博客所在本地目录.blog/，右键单击选择Git bash，或 打开git bash 后cd E:\blog到博客所在目录

  • 输入git config --global user.name "你的用户名"并回车，我的用户名是MCEINS，输入git config --global user.name "MCEINS"

  • 再输入git config --global user.email "你的注册邮箱"并回车，我的注册邮箱是yan3101314@gmail.com那么我就输入git config --global user.email "yan3101314@gmail.com"

  • 输入ssh-keygen -t rsa -C "你的注册邮箱"并回车 生成一个密钥文件，我的注册邮箱是yan3101314@gmail.com那么我输入ssh-keygen -t rsa -C "yan3101314@gmail.com"（注意：这里的C为大写）。回车之后再连跟三个回车，红线那里表示密钥ssh文件所在的目录。

  • 进入ssh所在文件夹复制文件id_rsa.pub内容(用记事本打开，或推荐使用Typora一款markdown编辑器)

  • 回到GitHub点击Setting，找到Deploy keys，点击右上角的Add Deploy keys添加密钥，（注意勾选Allow write access）。

  • 回到git bash，输入ssh git@github.com并回车，!!!此处注意：可能会要你输入yes或者no，输入yse，出现You've successfully authenticated就代表成功了。
